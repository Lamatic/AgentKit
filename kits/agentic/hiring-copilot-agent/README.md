# Hiring Copilot Agent 🚀

## 💡 What it does
AI-powered recruiter assistant that automates resume screening and candidate evaluation.

## ⚡ Features
- Resume analysis
- Candidate scoring
- Hiring recommendations

## 🛠 Tech Stack
- Next.js
- TypeScript
- Lamatic AI
- Tailwind CSS
