# Travel Chatbot Flow

## Overview
This flow acts as an AI-powered travel assistant that interacts with users in a conversational manner. It helps users plan trips, explore destinations, and get personalized travel recommendations.

## Features
- Answers travel-related questions
- Suggests popular tourist destinations
- Recommends hotels and accommodations
- Provides travel tips, food suggestions, and cultural insights

## Use Case
Users can interact with the chatbot to:
- Discover places to visit in a destination
- Find the best hotels and stays
- Get travel recommendations based on their interests

## Example Queries

1. I want to visit Bali, suggest me some of the places to visit  
2. Give me some of the best hotels in Bali

## Example Input

{
  "message": "I want to visit Bali, suggest me some of the places to visit"
}
