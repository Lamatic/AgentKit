# Smart Travel Planner Flow

## Overview
This flow generates a complete travel itinerary based on user inputs such as destination, number of days, and budget. It provides structured, day-wise travel plans along with recommendations for places, food, and cultural experiences.

## Features
- AI-generated travel itineraries
- Day-wise trip planning
- Suggestions for places to visit with images and maps
- Local food and cultural recommendations
- Budget-aware travel planning

## Use Case
This flow is useful for:
- Planning trips quickly and efficiently
- Getting structured itineraries for any destination
- Exploring travel highlights, food, and culture in one place

## Example Input

{
  "destination": "Bali",
  "number_of_days": 5,
  "budget": "1000",
  "travel_style": "nature"
}

## Example Output
A detailed travel plan including:
- Introduction of the destination
- Best time to visit
- Highlights and attractions
- Food recommendations
- Day-wise itinerary with activities and stay suggestions
