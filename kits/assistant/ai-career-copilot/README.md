# AI Career Copilot

An AI-powered career assistant that analyzes resumes and provides personalized career guidance including skill analysis, job recommendations, learning roadmaps, project suggestions, and interview preparation.

## Features

- 🎯 **Skill Analysis**: Automatically extract and analyze skills from resumes
- 📊 **Readiness Score**: Get a score indicating career readiness for target domains
- 💼 **Job Role Recommendations**: Discover suitable job roles based on your skills
- 🗺️ **Learning Roadmap**: Get a step-by-step learning path to fill skill gaps
- 🚀 **Project Suggestions**: Practical projects to build portfolio and skills
- 💬 **Interview Questions**: Practice questions tailored to your target role

## Prerequisites

- Node.js 18+ and npm
- Lamatic.ai account with API access
- Lamatic flow exported (included in `flows/` directory)
