# Internal Assistant

Build an internal chatbot that finds precise answers within your company’s knowledge base.

## Type
bundle

## What This Does
Build an internal chatbot that finds precise answers within your company’s knowledge base.

## Flows
- **crawling-indexation**: Crawling Indexation
- **gdrive**: Google Drive Indexation
- **gsheet**: GSheet Indexation
- **knowledge-chatbot**: Contextual api to answer queries with knowledge from Content
- **onedrive**: Onedrive
- **postgres**: Postgres Indexation
- **s3**: S3 Indexation
- **scraping-indexation**: Scraping Indexation
- **sharepoint**: Sharepoint Indexation
- **slack-assistant**: Contextual Slack assistant with knowledge from Content
- **teams-assistant**: Contextual Microsoft teams bot with knowledge from Content
