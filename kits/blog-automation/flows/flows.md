# Flows

Automate blog post generation and publishing via webhooks or schedules. Includes SEO optimization and CMS integration steps.

## blog-drafting
Generates a professional blog post based on topic, keywords, and instructions.
**Nodes:** API Request → Generate Blog Draft → API Response

## blog-publish
Publishes the optimized blog post to WordPress or another CMS.
**Nodes:** API Request → Publish to WordPress → API Response

## blog-seo
Optimizes a blog post draft for SEO using target keywords.
**Nodes:** API Request → SEO Optimize Content → API Response
