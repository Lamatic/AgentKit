# Flows

Extract valuable insights from documents and unstructured information at scale.

## document-parsing-etl (mandatory)
**Nodes:** API Request → Branching → Extract from File → Collate Document Pages → Generate Requirements → Parse JSON → Loop → Chunking → Extract Chunks → Vectorize → Generate JSON → Transform Metadata → Index → Loop End → API Response

## chatbot-widget (mandatory)
**Nodes:** Chat Widget → Variables → RAG → Chat Response
**Requires:** data-source
