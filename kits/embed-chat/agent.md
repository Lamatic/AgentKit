# Embedded Chat

It uses intelligent workflows to index PDFs and webpages, then provides an interactive chat interface where users can ask questions about their documents through a modern Next.js interface.

## Flows
- **embedded-chatbot-chatbot**: Select the vector database to be queried.
- **embedded-chatbot-pdf-indexation**: Select the model to convert the texts into vector representations.
- **embedded-chatbot-resource-deletion**: Select the vector database where the action will be performed.
- **embedded-chatbot-websites-indexation**: Select the credentials for crawler authentication.
