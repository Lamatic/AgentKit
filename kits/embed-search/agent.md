# Embedded Search

It uses intelligent workflows to index PDFs and webpages, then provides an interactive chat interface where users can ask questions about their documents through a modern Next.js interface.

- **embedded-search-pdf-indexation**: Select the model to convert the texts into vector representations.
- **embedded-search-resource-deletion**: Select the vector database where the action will be performed.
- **embedded-search-search**: No description
- **embedded-search-websites-indexation**: Select the credentials for crawler authentication.
