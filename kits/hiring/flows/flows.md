# Flows

It uses intelligent workflows to analyze resumes, match candidates to job requirements, and provide detailed hiring recommendations through a modern Next.js interface.

## automation-hiring
Select the credentials for Gmail authentication. Required to access the Gmail API.
**Nodes:** API Request → Send Acknowledgement → Prepare Receipt Email → API Response → Collate Resume Contents → Extract Resume → Evaluate Candidate → Condition → Prepare Selection Mail → Prepare Rejection Mail → Send Interview Mail → Send Rejection Mail
