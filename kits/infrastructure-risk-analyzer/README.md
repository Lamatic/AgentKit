# Local Infrastructure Risk Analyzer

An AI agent designed to ingest municipal project blueprints, public tender descriptions, or infrastructure proposals to evaluate environmental impact, sentiment, and project delivery roadblocks.

## How it works
1. **Input:** Takes a project proposal description or text documentation.
2. **Analysis:** Orchestrates workflow nodes to extract risk parameters and cross-reference potential local impacts.
3. **Output:** Formats a clean, actionable **Stakeholder Risk Matrix**.
