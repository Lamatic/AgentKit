# AgentKit Collection (2 flows)

## Agent Overview

The **Nihongo Flow AgentKit** is an AI-powered contextual Japanese learning assistant. It is designed to generate highly customized Japanese reading materials and reading comprehension assessments dynamically. By shifting away from static textbooks, this agent allows learners to study Japanese using scenarios that are immediately relevant to their interests or daily lives, bound by specific proficiency metrics.

## Purpose

The primary purpose of this kit is to bridge the gap between vocabulary memorization and natural language acquisition. It provides:
1. **Contextual Immersion:** Generates stories, dialogues, or passages based on a user-defined context (e.g., "Buying a train ticket").
2. **Targeted Practice:** Seamlessly integrates specific user-provided vocabulary words into the generated text.
3. **Automated Assessment:** Tests the user's comprehension of the newly generated text across multiple linguistic dimensions (Grammar, Vocabulary, Kanji, and Context).

## Flow Descriptions

### 1. Lesson Flow

Generates a complete Japanese reading lesson based on JLPT level and user context.
*   **Input Constraints:** JLPT Level (N5-N1), Context/Topic (String), Target Words (Array of Strings).
*   **Output Structure:** Returns a structured JSON payload containing:
    *   `original`: The generated Japanese text (with `Kanji[furigana]` formatting).
    *   `romanji`: Pronunciation guide.
    *   `translation`: English translation of the text.
    *   `dictionary`: A contextual glossary of key grammar points and vocabulary used in the text.

### 2. Quiz Flow

Consumes the text generated by the Lesson Flow to create a customized reading comprehension quiz.
*   **Input Constraints:** JLPT Level, Context, Source Text, and specific Question Counts (`grammar`, `vocabulary`, `kanji`, and `context` [Comprehension]).
*   **Output Structure:** Returns a structured JSON object containing a `Questions` array. Each question object inside the array contains:
    *   `question_id` & `question_type`
    *   `points` (weight of the question)
    *   `text` (the question prompt)
    *   `options` (multiple choice options a-d)
    *   `answer` (the correct prefix, e.g., "(b)")

## Guardrails

To ensure high-quality educational content, the agent operates within the following guardrails:
*   **Level Constraint:** Grammar structures and Kanji usage must strictly align with the requested JLPT level (e.g., N5 text should not contain N2 grammar or un-annotated complex Kanji).
*   **JSON Schema Enforcement:** Output is strictly constrained to the predefined JSON schema to prevent frontend parsing errors.
*   **Furigana Formatting:** All Kanji must be paired with its reading using the `Kanji[furigana]` bracket format to allow programmatic HTML `<ruby>` tag conversion.
