# Smart Trip Itinerary Planner

## Overview
Smart Trip Itinerary Planner is an AI-powered travel assistant that creates personalized travel plans based on destination, budget, trip duration, and user preferences.

## Features
The agent generates:

- Trip overview
- Day-wise itinerary
- Budget estimation
- Hotel recommendations
- Local transportation options
- Food suggestions
- Packing checklist
- Travel tips

## Inputs
The user provides:

- Destination
- Trip duration
- Budget range
- Number of travelers
- Travel preferences and interests

## Output
The agent returns a complete travel itinerary including recommended activities, accommodations, transportation options, estimated costs, and useful travel advice.

## Purpose
The goal is to simplify travel planning and provide users with a complete itinerary in a single interaction.
