# Gym AI Coach Agent

## Problem
Many beginners struggle to create proper diet and workout plans.

## Solution
An AI-powered agent that generates personalized fitness plans.

## Features
- Budget-friendly Indian diet
- Workout planning
- Protein estimation
- Structured output

## Setup
1. Add your API key (Gemini or other)
2. Import flow into Lamatic
3. Run the agent

## Usage
Enter:
Weight, Goal, Budget, Diet

Get:
Complete fitness plan instantly
