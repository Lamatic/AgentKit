# Gym AI Coach Flow

## Overview
This flow generates personalized fitness plans including diet, workout, and protein suggestions.

## Input
User provides:
- Weight
- Goal
- Budget
- Diet type

## Output
- Diet Plan
- Workout Plan
- Protein Sources
- Tips

## How It Works
1. User input is captured via chat widget
2. AI processes input using prompt engineering
3. Structured output is generated

## Use Case
Helps beginners create fitness plans easily without expert knowledge.
