# Multi Vector Search

## About This Flow

This flow integrates vector search into your website, allowing you to combine results from multiple vector databases and run parallel searches. It then consolidates the results and returns them to users.

This flow includes **14 nodes** working together to process data efficiently.

## Flow Components

This workflow includes the following node types:
- searchTriggerNode
- codeNode
- branchNode
- addNode
- searchNode
- searchResponseNode

## Files Included

- **config.json** - Complete flow structure with nodes and connections
- **inputs.json** - Private inputs requiring configuration
- **meta.json** - Flow metadata and information
